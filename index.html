Qué es webpack?

Webpack es una herramienta que nos permite preparar nuestro código para llevarlo a producción (module bundler)

Webpack nos permite trabajar con

  • HTML
  • CSS
  • JavaScript
  • Archivos estáticos
  • Imágenes
  • Fuentes

Tambien nos permite tener un modo en desarrollo para nuestros proyectos para hacer pruebas Nacio en el 2012, desde ese entonces varias empresas lo usan como ser

  • Twitter
  • Instagram
  • PayPal

También nos permite

  • Gestionar dependencias
  • Ejecutar tareas
  • Conversión de archivos
  • Nos permite trabajar en módulos
  • Permitiéndonos tener un código separado en desarrollo, pero en producción en una fuente

Webpack permite tener módulos de JS en formato

  • AMD
  • Common JS
  • ES15
